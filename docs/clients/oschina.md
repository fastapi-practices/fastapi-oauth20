# 开源中国

用于接入 OSChina 第三方登录。

## 申请步骤

1. 登录 [开源中国](https://www.oschina.net/)。
2. 从页脚进入开放平台或文档入口。

   ![settings.png](../public/images/oschina/settings.png)

3. 创建应用。

   ![new.png](../public/images/oschina/new.png)

4. 填写信息并确认添加。

   ![save.png](../public/images/oschina/save.png)

5. 创建完成后记录密钥。

   ![secrets.png](../public/images/oschina/secrets.png)

## 集成需要

- `应用 ID`
- `应用私钥`
- `回调地址`
